##  File Overview

| File | Topic | Description |
|------|-------|------------|
| `index.html` | **Full Stack Development** | Explains front‑end, back‑end, and database layers with a simulated request lifecycle. |
| `CET138-page-2-HTML.html` | **HTML** | Covers headings, lists, tables, forms, multimedia, semantics, and more. |
| `CET138-page-3-CSS.html` | **CSS** | Demonstrates selectors, box model, Flexbox, Grid, animations, variables, and responsive design. |
| `CET138-page-4-Bootrstrap.html` | **Bootstrap 5** | Showcases grid, buttons, alerts, cards, navbar, modals, forms, and utilities using CDN. |
| `CET138-page-5-Javascript.html` | **JavaScript** | Interactive demos for variables, functions, conditionals, loops, arrays, objects, DOM manipulation, events, and async code. |

Every page includes **live demos**, **code snippets**, and a sticky navigation bar to move seamlessly between topics.
